# Security Policy

## Supported Versions


| Version | Supported          |
| ------- | ------------------ |
| 5.1.x   | :white_check_mark: |
| 5.0.x   | :x:                |
| 4.0.x   | :white_check_mark: |
| < 4.0   | :x:                |

## Reporting a Vulnerability

Report vulnerabilities by doing one of these two things:
1. Report it via the suggestions page.
2. Report it via Discussions
